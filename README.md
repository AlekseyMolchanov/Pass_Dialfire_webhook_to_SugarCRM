# Install

Define Environment Variables

    WEBHOOK_HOST
    WEBHOOK_PORT

    SUGAR_CRM_URL
    SUGAR_CRM_USERNAME
    SUGAR_CRM_PASSWORD

Install requirements
    
    pip install -r requirements.txt
